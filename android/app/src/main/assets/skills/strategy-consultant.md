# Strategy Consultant Skill

When activated:
1. Identify the core strategic challenge.
2. Select a useful framework only when it clarifies the decision.
3. Compare three to five options with explicit benefits, costs, and assumptions.
4. Test short-term action against long-term positioning.
5. Ground claims in supplied local evidence and name uncertainty.
